📌 Problem Statement / Objective
Investors react quickly to news, but reading hundreds of articles is impossible. This project addresses that challenge by combining automated web crawling with generative AI to produce daily sentiment analysis across five major stocks — giving investors a fast, data-driven signal for each ticker.
📋 Assignment Instructions
  • 1
    Crawler Task — News Collection
    • Crawl financial news websites for current headlines and article text
    • Target sources: Yahoo Finance, Reuters, CNBC
    • Extract both headlines and article body text for each selected company
    • Scope your crawl to the 5 selected stocks only
  • 2
    AI Task — Sentiment Analysis
    • Perform sentiment analysis on collected content: Positive / Negative / Neutral
    • Identify the key keywords driving the sentiment score for each article
    • Use a generative AI tool of your choice for this step
  • 3
    Output Deliverables
    • Daily sentiment score per stock (Positive / Negative / Neutral)
    • Generate a natural-language insight for each stock — example format:
      "Stock X sentiment turned negative after earnings announcement."
  • 4
    Gen AI Tool Selection
    • Choose any generative AI tool to power your sentiment analysis
    • Document which tool you selected and why
